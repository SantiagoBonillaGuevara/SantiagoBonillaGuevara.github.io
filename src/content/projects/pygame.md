# 👾 The Galaga 2.0 Your Grandma Never Got to Beat!
Remember **Galaga**? Well, this isn't exactly Galaga...  
🧨 It's Galaga on steroids with customization options so absurd you'll be asking yourself:  
**What if Mario fought hideous creatures like in Doom?**

---

## 🛠️ What Do You Need to Play?

Before diving into this retro-customizable gem, make sure you have the following:

### ✅ Requirements:
- Python 3.x (the newest version, if possible).
- The magic library called `pygame` installed. If you don't have it, cast this spell in your terminal:
```bash
pip install pygame
```
- Clone the repository:
```bash
git clone https://github.com/SantiagoBonillaGuevara/pyGame
```
- And inside, run the main.py file:
```bash
python main.py
```

> 🤓 **Nerd fact:** As you score points (every 50), the game gets harder.  
> Meaning you're going to lose anyway. But with style. 😎

### ☠️ Did You Die?
Don't worry (well, maybe a little).  
**When you lose all your lives, you go straight back to the main menu.**  
There you can rethink your life choices, change your style, lower the difficulty, or just cry quietly.
